# Vancouver Bus Locator

A functioning realtime bus locator. Clicking on the markers will give information about the bus' route, direction and destination. 

## Technologies used: 

* create-react-app
* react-redux
* redux-saga
* superagent
* styled-components
* react-map-gl
* semantic-ui
